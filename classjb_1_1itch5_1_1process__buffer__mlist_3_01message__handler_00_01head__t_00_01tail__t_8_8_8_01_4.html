Detailed Description

template<typename message_handler, typename head_t, typename... tail_t>
class jb::itch5::process_buffer_mlist< message_handler, head_t, tail_t... >

Partial specialization for a list with at least one message.

Recurse through the message list until it is parsed or handled by the specialization for an empty message type list

Template Parameters
message_handlera type that meets the interface defined in jb::itch5::message_handler_concept.
head_tthe first message type in the message type list
tail_tthe remaining message types in the message type list
