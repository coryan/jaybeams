Copy a device buffer to the host asynchronously.

Copies the range [first, last) from the device to the host, storing the results at result. The copy is started after all the events in wait have completed. The function returns a future object so you can wait for the copy to complete, and chain other operations.

Template Parameters
DeviceIteratorthe iterator type for the container on the device
HostIteratorthe iterator type for the container on the host
Parameters
firstthe beginning of the range to copy
lastthe end of the range to copy
resultthe beginning of the range to store the copy
queuecommand queue to perform the operation
waita list of events (potentially empty) to wait for before starting the copy
Returns
A Future, once fulfilled it points to the end of the result range.
