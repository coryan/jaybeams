1 ## JayBeams Tools
2 
3 The JayBeams tools serve both as (a) demonstrations of how to use the
4 library in complete programs, and (b) utilities to analyze market data
5 feeds to help in the design of the libraries and/or in writing notes
6 about them.
7 
8 These options for each program can be found by executing each program
9 with the --help option, for example, after you compile the tools you
10 could use:
11 
12  cd tools
13  ./itch5inside --help
14  Program Options:
15  --help ....
16 
17 All the program options can also be specified in a
18 [YAML](http://www.yaml.org/) file.
19 If you
20 find yourself typing the same options over and over you might consider
21 creating a configuration file instead. You can always override the
22 configurations from the file with command-line options.
23 
24 For example, if you are running the `itch5inside` program with the
25 same input file and varying other options you could create a
26 `itch5inside.yaml` with the following contents:
27 
28  input-file: /foo/bar/baz/02022015.NASDAQ_ITCH50.gz
29  enable-symbol-stats: 1
30  stats:
31  max-messages-per-microsecond: 100
32 
33 You can read more about how JayBeams setup configuration files in the
34 [documentation](doc/jb_configuration.html)
